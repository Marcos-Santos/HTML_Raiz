Trabalhando com Tabelas

Para tabela usamos os tag <table>:
<tr> para criar Linhas.
<td> para criar Colunas.

Titulo da Tabela
Cabeçalho da tabela
linha 1, célula 1 linha 1, célula 2
linha 2, célula 1 linha 2, célula 2
Rodapé final da tabela

Exemplo de comandos de uma tabela:
Tag Descrição
<table> Define uma tabela
<th> Define um cabeçalho de tabela
<tr> Define uma linha de tabela
<td> Define uma célula de tabela
<caption> Define um título de tabela
<colgroup> Define um grupo de colunas de tabela
<col> Define os valores de atributo para uma ou mais colunas em uma tabela
<thead> Define um cabeçalho de tabela
<tbody> Define o corpo de uma tabela
<tfoot> Defines o rodapé (footer) de uma tabela

Trabalhando com DIV e SPAN

O DIV <div> é usado para definir uma divisão ou uma seção do documento. Não é um
elemento semântico, no entanto, é comumente usado quando não há uma melho atribuição semântica
O SPAN <span> possuem a mesma finalidade, mais ele é um elemento de linha ou frase.
